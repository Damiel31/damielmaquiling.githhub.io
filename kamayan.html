KAMAYAN TUNA GRILLS

We develop a sales and inventory system for our client Kamayan Tuna Grills located in Davao City to help them manage their buisness properly

FIRST INTERACTION

After the admin sign up the admin can see the current status about the buisness in the Dashboard Tab.

Employees need to text through the secret key API to indicate thet they're on duty or absent.

MANAGING STOCKS

Since kamayan grills is very popular their stocks is instantly depleting fast so we add a status bar how many stocks are left.

The admin can add stock and notified through SMS API what stocks should buy.

REPORTS

We established a graphical representation of sales and cost reports for the system to track the flow of the buisness.

With this the admin can decide what should be done or what should be the priority to keep the buisness going.
