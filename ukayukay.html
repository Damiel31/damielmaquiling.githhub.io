UKAY-UKAY

Ukay-Ukay is an app that supports local brands in the philippines to sell their authentic clothing for the public.

FIRST INTERACTION

After the user signs up the user will choose what gender of clothing s/he would like to purchase

This way the user could save time and shop easily.

PICKING CLOTH

When picking cloths you like you can already see the actual image and what it looks like.

I make the texts bigger so that the user can surely decide what it wants.

PURCHASING

The user can use different card to purchase or even COD and coupon code is available for discounts.

Shipment address can be change in settings and to track or check orders it should be in users detail menu.
